# Doctor Appointment Booking System

## Overview

A full-stack Doctor Appointment Booking System that enables patients to search for doctors, book appointments, manage bookings, and access healthcare services through an intuitive web interface. The platform also provides an admin dashboard for managing doctors, appointments, and users.

## Live Demo

* User Portal:doctor-appointment-booking-6kuk.vercel.app
* Admin Portal:doctor-appointment-booking-tsmq.vercel.app

## Features

### Patient Features

* User Registration and Login
* Browse Available Doctors
* View Doctor Profiles
* Book Appointments
* Manage Appointments
* Secure Authentication
* Responsive User Interface

### Admin Features

* Admin Authentication
* Add and Manage Doctors
* View All Appointments
* Dashboard Analytics
* Manage Platform Data

## Installation

### Clone Repository

```bash
git clone <repository-url>
```

### Install Dependencies

Frontend:

```bash
cd frontend
npm install
```

Admin:

```bash
cd admin
npm install
```

Backend:

```bash
cd backend
npm install
```

### Run Locally

Backend:

```bash
npm start
```

Frontend:

```bash
npm run dev
```

Admin:

```bash
npm run dev
```
